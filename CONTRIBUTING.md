# Contributing

To contribute to ATGUI, You need to follow these simple steps:
## Environment Setup
> Have Python 3 ready for use.

This project uses python 3, so older versions simply won't work.

## Follow Simple Rules
- Do not use extra non-standard libraries.

> This project is pure python 3, no other external libraries allowed than the ones added by the original authors.

- Keep code style consistent.

> For the well being of all the contributors, don't commit weird or scary looking code

- PR Styling
> Don't forget to mention all changes in your PR description, keep it clean and consistent with the rest.

# Have fun

Of course don't forget to hydrate, rest up and stay healthy!
